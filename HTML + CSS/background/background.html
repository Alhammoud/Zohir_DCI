CSS hintergründe

Eigenschaften

Es gibt einige verschiedene Hintergrund Eigenschaften

background-image
Diese Eigenschaft ist die location von Bild, zum Beispiel h1{ background-image:url('http://website.com/img/background.jpg;') } oder relativer Pfad ohne https: h1{ background-image:url('img/eule.png'); }
background-color
Diese Eigenschaft ist die Farbe von Bild, zum Beispiel h1{ background-color:pink; } oder alternativ als h1{ background-color:#FFC0CB; }
background-image
Diese Eigenschaft ist die location von Bild, zum Beispiel h1{ background-image:url('http://website.com/img/background.jpg;') } oder relativer Pfad ohne https: h1{ background-image:url('img/eule.png'); }
background-repeat
Diese Eigenschaft beschreibt,wie oft der Hintergrund wiederholt werden soll, zum Beispiel einmalig h1{ background-repeat:no-repeat; } oder auf der Y-Achse: h1{ background-repeat:repeat-y; }
background-position:right
Diese Eigenschaft beschreibt,wie der Hintergrund positioniert werden soll, zum Beispiel einmalig h1{ background-position:right; }
background-position:center
Diese Eigenschaft beschreibt,wie der Hintergrund positioniert werden soll, zum Beispiel einmalig h1{ background-position:center; }
